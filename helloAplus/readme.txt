目录结构：
.
├── 3rdparty                 ## 需要映射的外部svn库、头文件、代码
├── bin                      ##编译完成之后生成的库或者可执行文件的存放目录
├── build                    ##编译相关的文件，编译时需要进入该目录编译
├── docs                     ## 相关文件的存放路径
├── include                  ## 需要依赖的公共头文件或开源头文件
├── lib                      ## 需要依赖的公共库或开源库
├── src                      ## source code
├ ├── src                  ## 用户实现的源码
├ ├── inc                  ## 用户实现的头文件
├
├── test                     ## 测试代码,生成的测试程序存放在bin文件夹中


编译说明：
    cd build;make all
